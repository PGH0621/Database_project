# 🎬 NewFlix  
SQLite 기반 영화 정보 웹 애플리케이션

## 1. 프로젝트 개요
NewFlix는 영화 데이터를 기반으로 한 웹 애플리케이션으로,  
SQLite 데이터베이스를 활용하여 영화 정보 조회, 사용자별 찜 기능, 시청 기록 저장 기능을 구현한 프로젝트입니다.  
UI보다는 **데이터베이스 구조 설계와 로직 구현**에 중점을 두어 포스터 이미지와 동영상은 일부분만 구현 하였습니다.

---

## 2. 사용 기술
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- 기타: Jinja2 Template

---

## 3. 주요 기능
- 월별 박스오피스 영화 조회
- 전체 영화 목록 조회
- 영화 상세 정보 페이지 제공
- 사용자별 영화 찜하기 / 찜 취소 기능
- 사용자별 찜한 영화 목록 메인 페이지 표시
- 영화 재생 모달 팝업
- 시청 시간 기록 및 이어서 재생 기능
- 시청 기록이 없는 경우 처음부터 재생

---

## 4. 데이터베이스 구성
- **movies**: 영화 기본 정보 (순위, 관객 수, 매출 등)
- **movie_info**: 영화 상세 정보 (장르, 감독, 배우 등)
- **favorites**: 사용자별 찜 목록
- **watch_history**: 사용자별 영화 시청 기록 및 시청 시간
간- **users**
---

## 5. 프로젝트 목적
- SQLite를 활용한 관계형 데이터베이스 설계 경험
- Flask와 DB를 연동한 CRUD 로직 구현
- 사용자 행동(찜, 시청 기록)을 데이터로 관리하는 구조 이해
- 웹 서비스에서 데이터 흐름과 상태 관리 경험
- 실제 웹 서비스를 최대한 똑같이 따라 만들어보며 어떤 구조를 사용하는지 고민해보고, 이해하기
